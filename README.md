# 🕹️ PIXEL ART GALLERY — 80s Tecno-Minimalista

![License](https://img.shields.io/badge/license-MIT-green)
![Platform](https://img.shields.io/badge/platform-web-blue)
![Style](https://img.shields.io/badge/style-pixel--art-ff00ff)

**Galería de arte digital interactiva** con estética retro de los 80, diseñada para exhibir obras 1/1, colecciones, GIFs animados y videos MP4.  
Permite a los visitantes subir sus propias piezas, descargarlas, dejar comentarios y disfrutar de un ambiente sonoro inmersivo (lluvia cálida).

![Preview](https://via.placeholder.com/800x400/0a0a1a/00ff44?text=PIXEL+ART+GALLERY+PREVIEW)

---

## ✨ Características

- 🎨 **Diseño pixel-perfect**: fuentes bitmap, scanlines, neones y animaciones de tipeo.
- 🖼️ **Soporte multimedia**: imágenes, GIFs y videos MP4.
- ⬆️ **Upload por tarjeta**: cada obra puede ser reemplazada con archivos locales (drag & drop vía botón).
- ⬇️ **Download**: descarga la obra actual con un clic.
- 📝 **Notas colaborativas**: escribe y guarda comentarios en cada tarjeta (visibles para todos).
- 🔊 **Sonido ambiental**: lluvia suave generada con Web Audio API (se activa con botón).
- 🔗 **Footer con wallet Solana**: efecto de tipeo y dirección para donaciones.
- 🌌 **Fondo dinámico**: estrellas animadas en canvas.

---

## 🚀 Cómo usar

1. Clona el repositorio o descarga el archivo `index.html`.
2. Abre `index.html` en tu navegador favorito (Chrome, Firefox, Edge).
3. ¡Explora la galería! Puedes:
   - Hacer clic en **[UPL]** para subir tu propia imagen/video.
   - Usar **[DWL]** para descargar la obra actual.
   - Escribir una nota y presionar **[ENT]** para guardarla.
   - Activar el sonido con el botón **♪ SND**.

> No se requiere servidor local. Todo funciona en el lado del cliente.

---

## 🛠️ Tecnologías

- HTML5 / CSS3 (Grid, Flexbox, animaciones)
- JavaScript (ES6+)
- Web Audio API (generación de ruido marrón + convolución)
- Canvas API (fondo de estrellas)
- Google Fonts (`Press Start 2P`)
